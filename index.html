Sistema de Votaciones

Integrantes:
  • José Ismael Arias Reyes
  • Luis Alfredo López López
Objetivos
Objetivo General
  • Desarrollar un sistema de votación y escrutinio utilizando las nuevas tecnologías electrónica, siempre tomando en cuenta las leyes que rigen la transparencia de los procesos electorales, así como como también las disposiciones que realice el TSE (Tribunal Supremo Electoral), con este proyecto se logrará que todo el proceso de elección es más sencillo, práctico y seguro para los votantes.
Objetivos Específicos
  • Crear un sistema utilizando las tecnologías informáticas modernas para hacer que el proceso electoral sea más fácil y sencillo para el ciudadano al momento de ejercer el sufragio.

  • Utilizar las nuevas tecnologías para que el votante obtenga mayor facilidad a la hora de ejercer el sufragio y que este seguro de que el proceso será legal.
Problemática

El TSE (Tribunal Supremo Electoral), mantiene un sistema de votaciones actual que posee un mal funcionamiento y un mal manejo de parte de las autoridades que son las encargadas de su desarrollo en cuanto a las elecciones, en el actual sistema de votación existen muchos contratiempos por eso es que existen demasiados votos que son anulados por la falta de información que brinda el tribunal a las personas votantes, La falta de seguridad a la hora de realizar el Escrutinio de las actas, y no dan a conocer el resultado a tiempo por la mala organización o por la falta de personal adecuado para realizar este proceso.
